ESTRUTURA
Model - onde fica as table
Controller - onde fica as function
View - Paginas de PHP e de HTML
Route - Onde fica os caminhos e apontando para as view
.env --local onde fica o nome, user e senha do BD


Passos

- Pelo CMD criar um projeto 
composer create-project laravel/laravel nome_projeto

- Criar controller
php artisan make:controller NomeController


- Criar model
php artisan make:model NomeModel -m

- Criar a migration 
conferir sintaxe de criar migration
https://laravel.com/docs/9.x/migrations#column-method-string

- Rodar projeto
php artisan serve

- Migrar
php artisan migrate
